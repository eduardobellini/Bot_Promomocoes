# Regras de Negócio Testadas

### 1.normalize_text: Testa a normalização de texto, removendo espaços extras e decodificando entidades HTML.


### 2.parse_price: Testa a extração de preços de strings em formatos brasileiros (como "R$ 1.500,00").


### 3.looks_like_gpu: Testa se um título de produto parece ser uma GPU, verificando padrões incluídos/excluídos.
