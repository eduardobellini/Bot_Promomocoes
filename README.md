# Regras de Negócio Testadas

### 1.normalize_text: Testa a normalização de texto, removendo espaços extras e decodificando entidades HTML.


### 2.parse_price: Testa a extração de preços de strings em formatos brasileiros (como "R$ 1.500,00").


### 3.looks_like_gpu: Testa se um título de produto parece ser uma GPU, verificando padrões incluídos/excluídos.

O site consiste em achar placas de videos que enviará mensagens para o bot no telegram e ele avisará se tiver alguma promoção por lá(telegram) do aliexpress, magazine luiza, pichau, shoppe e mercado livre. O QA deverá usar esse comando para verificar o código com o pytest (& "C:/Program Files/Python314/python.exe" -m pytest test_bot.py -v)
